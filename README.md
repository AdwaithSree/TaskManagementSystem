# 🚀 Task Management System

A modern Full Stack Task Management application built using **React.js** and **Django REST Framework**.

## 📌 Features

- 🔐 JWT Authentication
- 📝 Create Tasks
- ✏️ Edit Tasks
- 🗑️ Delete Tasks
- 📊 Analytics Dashboard
- 🔔 Browser Notifications
- 🔎 Search & Filter
- 📅 Deadline Tracking
- 📱 Responsive UI
- ✅ Backend Validation

---

## 🛠️ Tech Stack

### Frontend
- React.js
- React Router
- Axios
- React Toastify
- Recharts
- CSS

### Backend
- Django
- Django REST Framework
- JWT Authentication
- SQLite
